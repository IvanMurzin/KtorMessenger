Api Docs

All authorization occurs through the jwt token using Bearer header

* ### post /login<br>

request:

```json
{
  "email": "myemail@domain.name",
  "userName": "User Name",
  "passwordHash": "some hash"
}
```

response:

```json
{
  "token": "jwt token"
}
```
